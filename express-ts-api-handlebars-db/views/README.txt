Ce répertoire comporte les "sous-vues" / "sous-templates" "Handlebars"
qui correspondent à des remplacements de la partie {{{body}}}
du template principal layouts/main.html
----
handlebars est une version améliorée de moustache .
handlebars peut être utilisé en "pur js" coté client/navigateur.
handlebars est ici utilisé coté serveur via "express-handlebars".
